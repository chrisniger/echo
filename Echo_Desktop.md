# Echo Desktop (Windows/macOS/Linux)

## Vision
Echo is a cross-platform AI desktop assistant for interviews, meetings, coding assessments, and professional collaboration.

## Technology Stack
- Tauri 2 + Rust
- React + TypeScript
- Tailwind CSS + shadcn/ui
- Zustand + TanStack Query
- SQLite + SQLCipher
- Whisper.cpp / Faster-Whisper
- CPAL/WASAPI/CoreAudio/PipeWire
- Auto-updater
- Sentry
